# AI编程交流社区 - 项目总计划

## 一、项目概述

打造一个面向开发者的AI编程交流社区，提供论坛讨论、代码分享、技术问答、项目展示、技术博客等核心功能，构建完整的开发者交流生态。

## 二、技术栈

| 层级 | 技术选型 |
|------|----------|
| 前端 | React 18 + TypeScript + Vite |
| 后台管理 | React 18 + Ant Design Pro |
| 后端 | Node.js + Express + TypeScript |
| 数据库 | PostgreSQL (Prisma ORM) + Redis |
| 认证 | JWT + OAuth2.0 |
| 实时通信 | Socket.IO |
| 文件存储 | 阿里云OSS / 七牛云 |
| 搜索引擎 | Elasticsearch |
| 部署 | Docker + Nginx |

## 三、设计规范

- 主色调：科技蓝配色方案（#2563EB 主色 / #3B82F6 次色 / #60A5FA 辅助色 / #EFF6FF 背景色 / #1D4ED8 深色），坚决不使用紫色
- UI风格：调用 UI/UX PRO MAX skills 设计，追求高端精致的视觉体验
- 坚决不使用任何表情包和emoji
- 响应式设计，适配桌面端和移动端
- 暗色模式支持

## 八、前后端接口一致性规范（必须严格遵守）

- 所有接口统一使用 `/api/v1/` 前缀，前端和后端必须完全一致
- 前端每个模块的"接口对接"部分，与后端对应模块的"接口设计"部分必须路径、方法完全对应
- 任何一端修改接口路径或方法，必须同步更新另一端的文档和代码
- 新增接口时前后端文档必须同时添加，不允许单端新增
- 以下为前后端模块对应关系，开发时必须对照检查：

| 前端模块 | 后端模块 |
|---------|---------|
| 02-用户认证系统 | 03-用户认证接口 |
| 03-论坛讨论模块 | 04-论坛帖子接口 |
| 04-代码分享模块 | 05-代码分享接口 |
| 05-技术问答模块 | 06-问答系统接口 |
| 06-项目展示模块 | 07-项目展示接口 |
| 07-技术博客模块 | 08-博客系统接口 |
| 08-个人中心模块 | 14-用户资料接口 |
| 09-评论互动模块 | 09-评论系统接口 |
| 10-消息通知模块 | 10-消息通知接口 |
| 11-搜索系统模块 | 11-搜索服务接口 |
| 12-标签分类模块 | 15-标签系统接口 |
| 13-排行榜与积分 | 16-排行榜与积分接口 |
| 所有涉及图片上传 | 12-文件上传接口 |
| 后台管理全部模块 | 13-管理后台接口 |
