# 模块10：数据统计与分析模块

> 重要：本模块的接口路径必须与后端管理后台接口（模块13）保持完全一致，任何变更需前后端同步更新。

## 模块概述
提供全站运营数据的深度分析和可视化展示，帮助运营人员了解社区状态和发展趋势。

## 设计要求
- 科技蓝配色方案（坚决不使用紫色）
- 调用 UI/UX PRO MAX skills 设计
- 图表类型丰富，交互性强
- 禁止使用任何表情包和emoji

## 功能清单

### 1. 用户数据分析
- 注册用户增长趋势（日/周/月/年）
- 活跃用户分析（DAU/WAU/MAU）
- 用户留存率分析
- 新用户转化漏斗
- 用户地域分布
- 用户来源渠道分析

### 2. 内容数据分析
- 各类内容发布趋势
- 内容质量分析（平均点赞、回复、收藏数）
- 热门内容排行
- 内容分类分布
- 标签使用热度分析

### 3. 互动数据分析
- 评论/回复数量趋势
- 点赞/收藏数量趋势
- 用户活跃时段热力图
- 问答回复率和解决率

### 4. 流量数据分析
- PV/UV趋势
- 页面访问排行
- 平均停留时间
- 跳出率分析

### 5. 报表导出
- 自定义时间范围
- 数据导出（CSV/Excel/PDF）
- 定时报表（每日/每周自动发送）

## 组件结构
```
src/pages/Analytics/
├── PrivateMessageMonitor.tsx # 私信监控
├── UserAnalytics.tsx          # 用户分析
├── ContentAnalytics.tsx       # 内容分析
├── InteractionAnalytics.tsx   # 互动分析
├── TrafficAnalytics.tsx       # 流量分析
├── ReportExport.tsx           # 报表导出
└── components/
    ├── TrendChart.tsx          # 趋势图
    ├── PieChart.tsx            # 饼图
    ├── HeatmapChart.tsx        # 热力图
    ├── FunnelChart.tsx         # 漏斗图
    ├── RankingTable.tsx        # 排行表
    ├── DateRangePicker.tsx     # 日期范围选择
    └── ExportButton.tsx        # 导出按钮
```

## 接口对接
```
GET /api/v1/admin/analytics/users          # 用户分析数据
GET /api/v1/admin/analytics/content        # 内容分析数据
GET /api/v1/admin/analytics/interactions   # 互动分析数据
GET /api/v1/admin/analytics/traffic        # 流量分析数据
GET /api/v1/admin/analytics/retention      # 留存率数据
GET /api/v1/admin/analytics/funnel         # 漏斗数据
GET /api/v1/admin/analytics/private-messages # 私信监控数据
POST /api/v1/admin/analytics/export        # 导出报表
```

## 验收标准
- 图表渲染流畅，大数据量不卡顿
- 时间范围切换数据刷新迅速
- 数据导出格式正确
- 所有图表支持交互（悬浮显示详情、缩放等）
- 数据统计口径准确一致
