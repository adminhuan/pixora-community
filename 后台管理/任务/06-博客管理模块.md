# 模块06：博客管理模块

> 重要：本模块的接口路径必须与后端管理后台接口（模块13）保持完全一致，任何变更需前后端同步更新。

## 模块概述
管理用户发布的博客文章，包括推荐、审核、编辑、删除等操作。

## 设计要求
- 科技蓝配色方案（坚决不使用紫色）
- 调用 UI/UX PRO MAX skills 设计
- 禁止使用任何表情包和emoji

## 功能清单

### 1. 博客列表管理
- 文章列表（标题、作者、分类、状态、发布时间、阅读量）
- 搜索筛选（关键词、分类、作者、时间、状态）
- 设为推荐/取消推荐
- 设为首页轮播
- 删除文章
- 查看文章详情

### 2. 系列文章管理
- 系列列表
- 编辑系列信息
- 删除系列
- 调整系列内文章顺序

### 3. 分类管理
- 博客分类列表
- 添加/编辑/删除分类
- 分类排序

## 组件结构
```
src/pages/BlogAdmin/
├── BlogManage.tsx             # 博客管理
├── SeriesManage.tsx           # 系列管理
├── BlogCategoryManage.tsx     # 分类管理
└── components/
    ├── BlogTable.tsx           # 博客表格
    ├── BlogFilters.tsx         # 筛选器
    └── SeriesEditor.tsx        # 系列编辑
```

## 接口对接
```
GET    /api/v1/admin/blogs                # 博客列表
PUT    /api/v1/admin/blogs/:id/recommend  # 推荐
PUT    /api/v1/admin/blogs/:id/banner     # 设为轮播
DELETE /api/v1/admin/blogs/:id            # 删除
GET    /api/v1/admin/blog-series          # 系列列表
PUT    /api/v1/admin/blog-series/:id      # 编辑系列
DELETE /api/v1/admin/blog-series/:id      # 删除系列
GET    /api/v1/admin/blog-categories      # 分类列表
POST   /api/v1/admin/blog-categories      # 创建分类
PUT    /api/v1/admin/blog-categories/:id  # 编辑分类
DELETE /api/v1/admin/blog-categories/:id  # 删除分类
```

## 验收标准
- 推荐文章在前端首页正确展示
- 文章管理操作不影响评论数据
- 分类删除有关联文章提示
- 系列文章排序正确
