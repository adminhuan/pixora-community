# 模块01：管理后台布局与仪表盘

> 重要：本模块的接口路径必须与后端管理后台接口（模块13）保持完全一致，任何变更需前后端同步更新。

## 模块概述
后台管理系统的整体框架和仪表盘首页，展示核心运营数据和快捷操作入口。

## 设计要求
- 科技蓝配色方案（坚决不使用紫色）（与用户端风格统一但有管理感）
- 调用 UI/UX PRO MAX skills 设计
- 仪表盘数据可视化图表清晰直观
- 禁止使用任何表情包和emoji

## 功能清单

### 1. 后台布局
- 左侧菜单栏（可折叠）
  - 仪表盘
  - 用户管理
  - 内容审核
  - 论坛管理
  - 问答管理
  - 博客管理
  - 项目管理
  - 评论管理
  - 标签管理
  - 数据统计
  - 系统配置
  - 运营工具
- 顶部信息栏（管理员信息、通知、退出）
- 面包屑导航
- 内容区域

### 2. 仪表盘数据概览
- 今日新增用户数
- 今日新增内容数（帖子、博客、问题、代码）
- 活跃用户数（日/周/月）
- 待审核内容数量
- 待处理举报数量
- 总注册用户数

### 3. 数据图表
- 用户增长趋势图（近7天/30天/90天）
- 内容发布趋势图
- 用户活跃时段分布图
- 各板块内容占比饼图
- 热门标签排行

### 4. 快捷操作
- 快速审核入口
- 举报处理入口
- 发布系统公告
- 近期操作日志

## 组件结构
```
src/
├── layouts/
│   ├── AdminLayout.tsx        # 管理后台主布局
│   ├── SideMenu.tsx           # 左侧菜单
│   ├── AdminHeader.tsx        # 顶部栏
│   └── Breadcrumb.tsx         # 面包屑
├── pages/
│   └── Dashboard/
│       ├── index.tsx           # 仪表盘主页
│       ├── StatsCards.tsx      # 数据概览卡片
│       ├── GrowthChart.tsx     # 增长趋势图
│       ├── ActivityChart.tsx   # 活跃分布图
│       ├── ContentPieChart.tsx # 内容占比图
│       └── RecentActions.tsx   # 近期操作
└── router/
    └── index.tsx               # 后台路由配置
```

## 技术选型
- Ant Design Pro：后台UI框架
- ECharts 或 Recharts：数据可视化图表
- dayjs：日期处理

## 接口对接
```
GET /api/v1/admin/dashboard/stats        # 概览统计数据
GET /api/v1/admin/dashboard/trends       # 趋势数据
GET /api/v1/admin/dashboard/activity     # 活跃数据
GET /api/v1/admin/dashboard/pending      # 待处理项
GET /api/v1/admin/logs/recent            # 近期操作日志
```

## 验收标准
- 仪表盘加载时间不超过3秒
- 图表数据准确，时间范围切换流畅
- 侧边菜单折叠/展开动画流畅
- 各入口跳转正确
- 响应式适配（最小支持1280px宽度）
