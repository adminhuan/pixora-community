# 模块12：运营工具模块

> 重要：本模块的接口路径必须与后端管理后台接口（模块13）保持完全一致，任何变更需前后端同步更新。

## 模块概述
提供日常运营所需的工具集，包括公告管理、推送管理、活动管理、操作日志等。

## 设计要求
- 科技蓝配色方案（坚决不使用紫色）
- 调用 UI/UX PRO MAX skills 设计
- 禁止使用任何表情包和emoji

## 功能清单

### 1. 公告管理
- 公告列表
- 发布公告（标题、内容、类型、是否置顶、显示时间范围）
- 编辑/删除公告
- 公告类型（系统公告、活动通知、版本更新、维护通知）

### 2. Banner管理
- 首页轮播Banner列表
- 添加Banner（图片、链接、排序、显示时间范围）
- 编辑/删除/排序Banner
- Banner启用/禁用

### 3. 推荐内容管理
- 推荐帖子管理
- 推荐博客管理
- 推荐项目管理
- 推荐位排序

### 4. 邮件群发
- 选择目标用户群体
- 邮件模板选择/编辑
- 发送预览
- 发送记录

### 5. 操作日志
- 管理员操作日志列表
- 按操作人、操作类型、时间筛选
- 操作详情查看
- 日志导出

### 6. 数据备份
- 手动触发备份
- 备份列表
- 备份下载
- 定时备份设置

## 组件结构
```
src/pages/Operations/
├── AnnouncementManage.tsx     # 公告管理
├── BannerManage.tsx           # Banner管理
├── RecommendManage.tsx        # 推荐管理
├── MassEmail.tsx              # 邮件群发
├── OperationLogs.tsx          # 操作日志
├── DataBackup.tsx             # 数据备份
└── components/
    ├── AnnouncementForm.tsx    # 公告表单
    ├── BannerUpload.tsx        # Banner上传
    ├── BannerSortable.tsx      # Banner拖拽排序
```

## 接口对接
```
GET    /api/v1/admin/announcements         # 公告列表
POST   /api/v1/admin/announcements         # 发布公告
PUT    /api/v1/admin/announcements/:id     # 编辑公告
DELETE /api/v1/admin/announcements/:id     # 删除公告
GET    /api/v1/admin/banners               # Banner列表
POST   /api/v1/admin/banners               # 添加Banner
PUT    /api/v1/admin/banners/:id           # 编辑Banner
DELETE /api/v1/admin/banners/:id           # 删除Banner
PUT    /api/v1/admin/banners/sort          # Banner排序
GET    /api/v1/admin/recommendations       # 推荐列表
PUT    /api/v1/admin/recommendations       # 更新推荐
POST   /api/v1/admin/email/send            # 发送邮件
GET    /api/v1/admin/email/history         # 邮件记录
GET    /api/v1/admin/logs                  # 操作日志
POST   /api/v1/admin/logs/export           # 导出日志
POST   /api/v1/admin/backup                # 创建备份
GET    /api/v1/admin/backup/list           # 备份列表
GET    /api/v1/admin/backup/:id/download   # 下载备份
PUT    /api/v1/admin/backup/schedule       # 定时备份设置
```

## 验收标准
- 公告发布后前端即时展示
- Banner排序拖拽流畅
- 邮件群发有进度显示
- 操作日志记录完整
- 备份功能稳定可靠
