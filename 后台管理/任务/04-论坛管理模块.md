# 模块04：论坛管理模块

> 重要：本模块的接口路径必须与后端管理后台接口（模块13）保持完全一致，任何变更需前后端同步更新。

## 模块概述
管理论坛板块、帖子内容、置顶精华等，维护论坛秩序和内容质量。

## 设计要求
- 科技蓝配色方案（坚决不使用紫色）
- 调用 UI/UX PRO MAX skills 设计
- 禁止使用任何表情包和emoji

## 功能清单

### 1. 板块/分类管理
- 分类列表（排序、启用/禁用）
- 添加/编辑/删除分类
- 分类排序拖拽
- 分类图标设置

### 2. 帖子管理
- 帖子列表（标题、作者、分类、状态、发布时间、浏览量、回复数）
- 搜索筛选（关键词、分类、时间、状态）
- 置顶/取消置顶
- 设为精华/取消精华
- 锁定帖子（禁止回复）
- 移动帖子（更换分类）
- 删除帖子
- 查看帖子详情

### 3. 回复管理
- 查看帖子回复列表
- 删除违规回复
- 折叠回复

## 组件结构
```
src/pages/ForumAdmin/
```

## 接口对接
```
GET    /api/v1/admin/categories           # 分类列表
POST   /api/v1/admin/categories           # 创建分类
PUT    /api/v1/admin/categories/:id       # 编辑分类
DELETE /api/v1/admin/categories/:id       # 删除分类
PUT    /api/v1/admin/categories/sort      # 分类排序
GET    /api/v1/admin/posts                # 帖子列表
PUT    /api/v1/admin/posts/:id/pin        # 置顶
PUT    /api/v1/admin/posts/:id/feature    # 精华
PUT    /api/v1/admin/posts/:id/lock       # 锁定
PUT    /api/v1/admin/posts/:id/move       # 移动
DELETE /api/v1/admin/posts/:id            # 删除
```

## 验收标准
- 分类排序拖拽流畅
- 帖子管理操作响应迅速
- 批量操作功能正常
- 移动帖子后分类关联正确
