# 模块03：论坛讨论模块

> 重要：本模块的接口路径必须与后端对应模块保持完全一致，任何变更需前后端同步更新。

## 模块概述
构建社区的核心论坛功能，支持帖子的发布、浏览、回复、点赞、收藏等完整交互流程。

## 设计要求
- 科技蓝配色方案（坚决不使用紫色）
- 调用 UI/UX PRO MAX skills 设计
- 帖子列表支持多种视图模式（列表/卡片）
- 禁止使用任何表情包和emoji

## 功能清单

### 1. 帖子列表页
- 分类筛选（技术讨论、经验分享、求助、闲聊等）
- 排序方式（最新、最热、精华）
- 标签筛选
- 分页/无限滚动加载
- 帖子预览卡片（标题、摘要、作者、时间、浏览量、回复数、点赞数）

### 2. 发帖功能
- 富文本 Markdown 编辑器
- 标题输入
- 分类选择
- 标签添加（已有标签搜索 + 自定义标签）
- 图片上传（拖拽 + 粘贴）
- 代码块插入（语法高亮预览）
- 草稿自动保存
- 发布前预览

### 3. 帖子详情页
- 帖子完整内容渲染（Markdown转HTML）
- 代码块语法高亮
- 作者信息卡片
- 点赞、收藏、分享按钮
- 回复列表（支持楼中楼回复）
- 举报功能
- 编辑/删除（作者权限）
- 相关帖子推荐

### 4. 互动功能
- 点赞/取消点赞
- 收藏/取消收藏
- 分享（复制链接、社交平台分享）
- 置顶/加精（管理员权限）

## 组件结构
```
src/pages/Forum/
├── ForumList.tsx           # 帖子列表页
├── ForumDetail.tsx         # 帖子详情页
├── CreatePost.tsx          # 发帖页面
├── EditPost.tsx            # 编辑帖子
└── components/
    ├── PostCard.tsx         # 帖子卡片
    ├── PostContent.tsx      # 帖子内容渲染
    ├── CategoryFilter.tsx   # 分类筛选
    ├── TagFilter.tsx        # 标签筛选
    └── SortBar.tsx          # 排序栏
```

## 接口对接
```
GET    /api/v1/posts              # 帖子列表（分页、筛选、排序）
GET    /api/v1/posts/:id          # 帖子详情
POST   /api/v1/posts              # 创建帖子
PUT    /api/v1/posts/:id          # 编辑帖子
DELETE /api/v1/posts/:id          # 删除帖子
POST   /api/v1/posts/:id/like     # 点赞
POST   /api/v1/posts/:id/favorite # 收藏
GET    /api/v1/posts/:id/related  # 相关帖子
POST   /api/v1/upload/image       # 上传图片（帖子内图片）
```

## 依赖库
- @uiw/react-md-editor 或 vditor：Markdown编辑器
- react-syntax-highlighter 或 prism.js：代码高亮
- react-markdown：Markdown渲染

- 代码块语法高亮准确
- 帖子列表加载流畅，支持筛选和排序
- 回复支持嵌套展示
- 图片上传成功率100%
