# 模块06：项目展示模块

> 重要：本模块的接口路径必须与后端对应模块保持完全一致，任何变更需前后端同步更新。

## 模块概述
为开发者提供展示个人项目的平台，支持项目截图、技术栈标注、在线演示链接、源码链接等。

## 设计要求
- 科技蓝配色方案（坚决不使用紫色）
- 调用 UI/UX PRO MAX skills 设计，项目卡片需要视觉冲击力
- 瀑布流或网格布局展示
- 禁止使用任何表情包和emoji

## 功能清单

### 1. 项目列表页
- 网格卡片展示（项目封面图 + 标题 + 简介 + 技术栈标签）
- 分类筛选（Web应用、移动端、工具类、游戏、AI项目等）
- 技术栈筛选
- 排序（最新、最热、星标最多）
- 搜索功能

### 2. 发布项目
- 项目名称
- 项目简介（一句话描述）
- 详细介绍（Markdown编辑器）
- 封面图上传
- 截图上传（最多6张，支持拖拽排序）
- 技术栈选择（多选标签）
- 在线演示链接（Demo URL）
- 源码链接（GitHub/Gitee等）
- 项目状态（开发中/已完成/维护中/已废弃）

### 3. 项目详情页
- 封面大图
- 项目信息概览
- 截图画廊（灯箱效果）
- 详细介绍内容渲染
- 技术栈标签展示
- 快捷操作按钮（在线预览、查看源码）
- 点赞、收藏、分享
- 评论讨论区
- 作者其他项目推荐

### 4. 项目评分
- 用户评分（1-5星）
- 综合评分展示
- 评价内容

## 组件结构
```
src/pages/Projects/
├── ProjectList.tsx           # 项目列表
├── ProjectDetail.tsx         # 项目详情
├── CreateProject.tsx         # 发布项目
├── EditProject.tsx           # 编辑项目
└── components/
    ├── ProjectCard.tsx        # 项目卡片
    ├── TechStackTags.tsx      # 技术栈标签
    ├── ScreenshotGallery.tsx  # 截图画廊
    ├── ProjectStatus.tsx      # 项目状态标识
    ├── RatingStars.tsx        # 评分组件
    └── DemoButton.tsx         # 演示按钮
```

## 接口对接
```
GET    /api/v1/projects              # 项目列表
GET    /api/v1/projects/:id          # 项目详情
POST   /api/v1/projects              # 创建项目
PUT    /api/v1/projects/:id          # 编辑项目
DELETE /api/v1/projects/:id          # 删除项目
POST   /api/v1/projects/:id/like     # 点赞
POST   /api/v1/projects/:id/favorite # 收藏
POST   /api/v1/projects/:id/rate     # 评分
GET    /api/v1/projects/:id/ratings  # 评分列表
GET    /api/v1/projects/:id/related  # 相关项目
POST   /api/v1/upload/image          # 上传图片（封面/截图）
```

## 验收标准
- 项目卡片视觉效果美观
- 截图画廊交互流畅
- 技术栈标签展示清晰
- 外部链接在新窗口打开
- 图片加载有占位效果和懒加载
