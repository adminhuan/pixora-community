# 模块07：技术博客模块

> 重要：本模块的接口路径必须与后端对应模块保持完全一致，任何变更需前后端同步更新。

## 模块概述
为开发者提供技术博客撰写和发布平台，支持长文创作、系列文章、专栏管理等功能。

## 设计要求
- 科技蓝配色方案（坚决不使用紫色）
- 调用 UI/UX PRO MAX skills 设计，注重阅读体验
- 文章阅读页面排版优雅，代码高亮清晰
- 禁止使用任何表情包和emoji

## 功能清单

### 1. 博客列表页
- 文章卡片（封面图 + 标题 + 摘要 + 作者 + 发布时间 + 阅读量）
- 分类导航（前端、后端、移动端、AI/ML、DevOps、数据库等）
- 标签筛选
- 排序（最新、最热、推荐）
- 搜索功能

### 2. 写博客
- Markdown编辑器（左右分栏实时预览）
- 标题输入
- 封面图上传
- 分类选择
- 标签添加
- 摘要编辑（自动提取 + 手动编辑）
- 草稿保存（自动 + 手动）
- 定时发布
- 系列文章关联（选择所属系列）

### 3. 文章详情页
- 文章内容渲染（优秀的排版样式）
- 目录导航（根据标题自动生成侧边目录）
- 阅读进度条
- 代码块语法高亮 + 一键复制
- 点赞、收藏、分享
- 评论区
- 上一篇/下一篇导航
- 系列文章目录（如果属于系列）
- 作者信息卡片
- 相关文章推荐

### 4. 专栏/系列文章
- 创建系列（系列名称、简介、封面）
- 系列文章排序管理
- 系列详情页（文章列表、总阅读量、关注数）

### 5. 草稿箱
- 草稿列表
- 继续编辑
- 删除草稿

## 组件结构
```
src/pages/Blog/
├── BlogList.tsx              # 博客列表
├── BlogDetail.tsx            # 文章详情
├── WriteBlog.tsx             # 写博客
├── EditBlog.tsx              # 编辑博客
├── DraftList.tsx             # 草稿箱
├── SeriesDetail.tsx          # 系列详情
└── components/
    ├── ArticleCard.tsx        # 文章卡片
    ├── MarkdownEditor.tsx     # Markdown编辑器
    ├── TableOfContents.tsx    # 目录导航
    ├── ReadingProgress.tsx    # 阅读进度条
    ├── SeriesNav.tsx          # 系列导航
    ├── CategoryNav.tsx        # 分类导航
    └── ArticleFooter.tsx      # 文章底部（上下篇、推荐）
```

## 接口对接
```
GET    /api/v1/blogs                  # 博客列表
GET    /api/v1/blogs/:id              # 文章详情
POST   /api/v1/blogs                  # 发布文章
PUT    /api/v1/blogs/:id              # 编辑文章
DELETE /api/v1/blogs/:id              # 删除文章
GET    /api/v1/blogs/drafts           # 草稿列表
POST   /api/v1/blogs/drafts           # 保存草稿
PUT    /api/v1/blogs/drafts/:id       # 更新草稿
DELETE /api/v1/blogs/drafts/:id       # 删除草稿
POST   /api/v1/blogs/drafts/:id/publish # 发布草稿
POST   /api/v1/blogs/:id/like         # 点赞
POST   /api/v1/blogs/:id/favorite     # 收藏
GET    /api/v1/series                  # 系列列表
POST   /api/v1/series                  # 创建系列
GET    /api/v1/series/:id              # 系列详情
PUT    /api/v1/series/:id              # 编辑系列
DELETE /api/v1/series/:id              # 删除系列
PUT    /api/v1/series/:id/order        # 调整文章顺序
POST   /api/v1/series/:id/follow       # 关注系列
POST   /api/v1/upload/image            # 上传图片（博客内图片）
```

## 验收标准
- 编辑器支持完整Markdown语法
- 文章阅读页面排版美观、代码高亮准确
- 目录导航定位准确，滚动跟随
- 草稿自动保存间隔不超过30秒
- 系列文章关联正确
