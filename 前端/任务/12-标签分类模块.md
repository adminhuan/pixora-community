# 模块12：标签分类模块

> 重要：本模块的接口路径必须与后端对应模块保持完全一致，任何变更需前后端同步更新。

## 模块概述
全站标签和分类体系，用于内容的分类管理和标签化组织，支持标签云展示、标签关注等。

## 设计要求
- 科技蓝配色方案（坚决不使用紫色）
- 调用 UI/UX PRO MAX skills 设计
- 标签云视觉效果突出
- 禁止使用任何表情包和emoji

## 功能清单

### 1. 标签体系
- 标签列表页（所有标签浏览）
- 标签云展示（按热度大小排列）
- 标签详情页（标签描述 + 该标签下的所有内容）
- 标签搜索
- 标签关注/取消关注

### 2. 分类体系
- 一级分类导航
- 二级分类筛选
- 分类内容列表

### 3. 标签选择器（通用组件）
- 输入搜索已有标签
- 创建新标签
- 最多选择标签数限制
- 热门标签推荐

## 组件结构
```
src/pages/Tags/
├── TagList.tsx                # 标签列表页
├── TagDetail.tsx              # 标签详情页
└── components/
    ├── TagCloud.tsx            # 标签云
    └── TagStats.tsx            # 标签统计

src/components/Tag/
├── TagSelector.tsx            # 标签选择器（通用）
├── TagBadge.tsx               # 标签徽章
└── CategoryNav.tsx            # 分类导航
```

## 接口对接
```
GET    /api/v1/tags                    # 标签列表
GET    /api/v1/tags/:id                # 标签详情
GET    /api/v1/tags/:id/contents       # 标签下的内容
GET    /api/v1/tags/hot                # 热门标签
POST   /api/v1/tags/:id/follow         # 关注标签
DELETE /api/v1/tags/:id/follow         # 取消关注
GET    /api/v1/categories              # 分类列表
GET    /api/v1/categories/:id/contents # 分类下的内容
```

## 验收标准
- 标签云视觉效果美观
- 标签选择器搜索响应快速
- 标签详情页内容聚合正确
- 分类导航层级清晰
