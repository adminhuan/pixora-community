# 模块08：个人中心模块

> 重要：本模块的接口路径必须与后端对应模块保持完全一致，任何变更需前后端同步更新。

## 模块概述
用户个人主页和设置中心，展示用户信息、发布内容、社交关系、积分等。

## 设计要求
- 科技蓝配色方案（坚决不使用紫色）
- 调用 UI/UX PRO MAX skills 设计个人主页
- 用户主页要有个性化视觉效果
- 禁止使用任何表情包和emoji

## 功能清单

### 1. 个人主页（公开）
- 用户头像 + 用户名 + 个性签名
- 背景封面图（可自定义）
- 基本信息（职位、公司、城市、GitHub链接、个人网站）
- 数据统计（帖子数、回答数、博客数、项目数、获赞数、粉丝数）
- 内容标签页
  - 帖子列表
  - 回答列表
  - 博客列表
  - 代码分享列表
  - 项目列表
  - 收藏列表
- 关注/粉丝列表
- 贡献热力图（类似GitHub）
- 成就徽章展示

### 2. 个人设置
- 基本信息编辑（头像、昵称、签名、个人简介）
- 账号安全（修改密码、绑定邮箱、绑定手机、第三方账号绑定）
- 隐私设置（是否公开邮箱、是否公开收藏）
- 通知设置（邮件通知、站内通知开关）
- 主题偏好（亮色/暗色模式）

### 3. 关注系统
- 关注/取消关注用户
- 关注列表
- 粉丝列表
- 互相关注标识

### 4. 个人数据
- 阅读历史
- 点赞历史
- 收藏夹管理（创建收藏夹、分类收藏）
- 积分明细

## 接口对接
```
GET    /api/v1/users/:id              # 用户信息
PUT    /api/v1/users/profile          # 更新个人信息
PUT    /api/v1/users/settings         # 更新设置
GET    /api/v1/users/:id/posts        # 用户帖子
GET    /api/v1/users/:id/blogs        # 用户博客
GET    /api/v1/users/:id/projects     # 用户项目
GET    /api/v1/users/:id/snippets     # 用户代码分享
GET    /api/v1/users/:id/followers    # 粉丝列表
GET    /api/v1/users/:id/following    # 关注列表
POST   /api/v1/users/:id/follow       # 关注
DELETE /api/v1/users/:id/follow       # 取消关注
GET    /api/v1/users/:id/favorites    # 收藏列表
GET    /api/v1/users/:id/contributions # 贡献数据
GET    /api/v1/users/:id/achievements  # 成就列表
GET    /api/v1/users/:id/answers       # 用户回答列表
POST   /api/v1/users/avatar            # 上传头像
POST   /api/v1/users/cover             # 上传封面图
POST   /api/v1/users/favorites/folders     # 创建收藏夹
PUT    /api/v1/users/favorites/folders/:id # 编辑收藏夹
DELETE /api/v1/users/favorites/folders/:id # 删除收藏夹
```

## 验收标准
- 个人主页内容丰富、布局美观
- 贡献热力图数据准确
- 关注/取关操作实时响应
- 设置修改即时生效
- 收藏夹分类管理功能完整
