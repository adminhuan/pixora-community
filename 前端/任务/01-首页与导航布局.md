# 模块01：首页与导航布局

> 重要：本模块的接口路径必须与后端对应模块保持完全一致，任何变更需前后端同步更新。

## 模块概述
构建社区的整体页面框架，包括顶部导航、侧边栏、首页内容展示区域，奠定整个前端应用的布局基础。

## 设计要求
- 科技蓝配色方案（坚决不使用紫色）
- 调用 UI/UX PRO MAX skills 进行界面设计
- 响应式布局，支持桌面端和移动端
- 支持暗色/亮色模式切换
- 禁止使用任何表情包和emoji

## 功能清单

### 1. 顶部导航栏
- Logo + 站点名称
- 主导航菜单：首页、论坛、问答、代码分享、项目展示、博客
- 搜索栏（全局搜索入口）
- 用户头像下拉菜单（登录/注册/个人中心/设置/退出）
- 消息通知图标（带未读数量徽标）
- 发布按钮（快速发帖/提问/分享）

### 2. 首页内容区域
- 轮播推荐区：精选内容、公告
- 热门帖子列表
- 最新问答列表
- 热门代码片段展示
- 推荐项目卡片
- 最新博客文章
- 活跃用户排行侧边栏
- 热门标签云

### 3. 页脚
- 关于我们、联系方式、隐私政策、用户协议
- 友情链接
- 备案信息

## 技术实现

### 路由配置
```
/                     首页
/forum                论坛
/qa                   问答
/code                 代码分享
/projects             项目展示
/blog                 博客
/user/:id             个人主页
/settings             设置
/notifications        消息通知
/search               搜索结果
```

### 组件结构
```
src/
├── layouts/
│   ├── MainLayout.tsx          # 主布局容器
│   ├── Navbar.tsx              # 顶部导航
│   ├── Sidebar.tsx             # 侧边栏
│   └── Footer.tsx              # 页脚
├── pages/
│   └── Home/
│       ├── index.tsx           # 首页
│       ├── HeroBanner.tsx      # 轮播推荐区
│       ├── HotPosts.tsx        # 热门帖子
│       ├── LatestQA.tsx        # 最新问答
│       ├── CodeHighlights.tsx  # 热门代码
│       └── TrendingTags.tsx    # 热门标签
├── router/
│   └── index.tsx               # 路由配置
└── styles/
    └── theme.ts                # 主题配置（可配置色值）
```

## 依赖库
- react-router-dom：路由管理
- styled-components 或 tailwindcss：样式方案
- framer-motion：动画效果
- react-icons：图标库

## 验收标准
- 导航栏在所有页面保持一致
- 响应式断点：桌面(1280px+)、平板(768px-1279px)、手机(767px-)
- 首页内容加载时间不超过2秒
- 暗色模式切换平滑无闪烁
- 所有交互元素有hover/active状态反馈

## 联动修复记录（2026-02-13）

- 首页 Hero 标题由硬编码改为读取站点配置 `siteName`，与导航栏保持一致。
- 首页模块统一接入真实接口数据：
  - 热门帖子：`GET /api/v1/posts?sort=hot`

- 首页列表项支持跳转到真实详情页（帖子/问答/组件/标签）。
- 首页关键点击位新增埋点上报：头图按钮、热门列表、侧栏排行/标签都会上报 `POST /api/v1/analytics/home-click`。

  - `detail_view`（详情浏览）
  - `detail_interaction`（点赞/收藏/评论等互动）
