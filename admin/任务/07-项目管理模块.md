# 模块07：项目管理模块

> 重要：本模块的接口路径必须与后端管理后台接口（模块13）保持完全一致，任何变更需前后端同步更新。

## 模块概述
管理用户发布的项目展示内容，包括审核、推荐、删除等操作。

## 设计要求
- 科技蓝配色方案（坚决不使用紫色）
- 调用 UI/UX PRO MAX skills 设计
- 禁止使用任何表情包和emoji

## 功能清单

### 1. 项目列表管理
- 项目表格（名称、作者、技术栈、状态、发布时间、星标数）
- 搜索筛选（关键词、技术栈、状态、时间）
- 设为推荐项目
- 设为首页展示
- 删除项目
- 查看项目详情

### 2. 项目审核
- 新项目审核
- 项目更新审核
- 外部链接安全检查

### 3. 项目分类管理
- 项目分类列表
- 添加/编辑/删除分类

## 组件结构
```
src/pages/ProjectAdmin/
├── ProjectManage.tsx          # 项目管理
├── ProjectCategoryManage.tsx  # 分类管理
└── components/
    ├── ProjectTable.tsx        # 项目表格
    ├── ProjectFilters.tsx      # 筛选器
    └── ProjectPreview.tsx      # 项目预览
```

## 接口对接
```
GET    /api/v1/admin/projects              # 项目列表
PUT    /api/v1/admin/projects/:id/recommend # 推荐
PUT    /api/v1/admin/projects/:id/feature   # 首页展示
DELETE /api/v1/admin/projects/:id           # 删除
GET    /api/v1/admin/project-categories     # 分类列表
POST   /api/v1/admin/project-categories     # 创建分类
PUT    /api/v1/admin/project-categories/:id # 编辑
DELETE /api/v1/admin/project-categories/:id # 删除
```

## 验收标准
- 项目管理操作正常
- 推荐项目在前端正确展示
- 分类管理完整
- 外部链接有安全提示
