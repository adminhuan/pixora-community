# 模块08：博客系统接口

> 重要：本模块接口路径必须与前端对应模块保持完全一致，任何变更需前后端同步更新。

## 模块概述
实现技术博客的后端接口，包括文章发布、草稿、系列文章等功能。

## 接口设计

```
# 博客文章
GET    /api/v1/blogs                    # 博客列表
GET    /api/v1/blogs/:id                # 文章详情
POST   /api/v1/blogs                    # 发布文章（需认证）
PUT    /api/v1/blogs/:id                # 编辑文章（需认证）
DELETE /api/v1/blogs/:id                # 删除文章（需认证）
POST   /api/v1/blogs/:id/like           # 点赞
POST   /api/v1/blogs/:id/favorite       # 收藏

# 草稿
GET    /api/v1/blogs/drafts             # 草稿列表（需认证）
POST   /api/v1/blogs/drafts             # 保存草稿（需认证）
PUT    /api/v1/blogs/drafts/:id         # 更新草稿
DELETE /api/v1/blogs/drafts/:id         # 删除草稿
POST   /api/v1/blogs/drafts/:id/publish # 发布草稿

# 系列文章
GET    /api/v1/series                   # 系列列表
GET    /api/v1/series/:id               # 系列详情（含文章列表）
POST   /api/v1/series                   # 创建系列（需认证）
PUT    /api/v1/series/:id               # 编辑系列
DELETE /api/v1/series/:id               # 删除系列
PUT    /api/v1/series/:id/order         # 调整文章顺序
POST   /api/v1/series/:id/follow        # 关注系列
```

## 业务逻辑
- 发布时自动计算阅读时间（按中文300字/分钟）
- 自动提取摘要（取前200字）
- Markdown转HTML服务端渲染
- 草稿自动保存不影响已发布版本
- 定时发布：到达指定时间自动发布（定时任务）
- 系列文章关联管理
