# 模块02：数据库模型设计

## 模块概述
设计和实现所有 PostgreSQL 数据模型（Prisma Schema），定义字段、索引、唯一约束、关联关系、枚举类型等。每个模型必须在 prisma/schema.prisma 中独立定义，禁止使用单表 JSON 存储。

## 数据模型清单

### 1. User（用户）
```typescript
{
  username: String,          // 用户名（唯一）
  email: String,             // 邮箱（唯一）
  phone: String,             // 手机号
  password: String,          // 密码（bcrypt加密）
  avatar: String,            // 头像URL
  coverImage: String,        // 封面图URL
  nickname: String,          // 昵称
  bio: String,               // 个人简介
  signature: String,         // 个性签名
  role: String,              // 角色（user/moderator/admin）
  status: String,            // 状态（active/muted/banned）
  mutedUntil: Date,          // 禁言截止时间
  bannedUntil: Date,         // 封禁截止时间
  position: String,          // 职位
  company: String,           // 公司
  city: String,              // 城市
  website: String,           // 个人网站
  github: String,            // GitHub链接
  points: Number,            // 积分
  level: Number,             // 等级
  followersCount: Number,    // 粉丝数
  followingCount: Number,    // 关注数
  postsCount: Number,        // 帖子数
  oauthProviders: [{         // 第三方登录
    provider: String,
    providerId: String
  }],
  notificationSettings: {},  // 通知设置
  lastLoginAt: Date,         // 最后登录时间
  createdAt: Date,
  updatedAt: Date
}
// 索引：username, email, phone, status, createdAt
```

### 12. 关系模型

#### Like（点赞）
```typescript
{
  user: 关联 User,
  targetType: String,
  targetId: String,
  createdAt: Date
}
// 索引：user+targetType+targetId（唯一）
```

#### Favorite（收藏）
```typescript
{
  user: 关联 User,
  targetType: String,
  targetId: String,
  folder: String,            // 收藏夹名称
  createdAt: Date
}
// 索引：user+targetType+targetId（唯一）
```

#### Follow（关注）
```typescript
{
  follower: 关联 User,
  following: 关联 User,
  createdAt: Date
}
// 索引：follower+following（唯一）
```

#### Vote（投票）
```typescript
{
  user: 关联 User,
  targetType: String,        // question/answer
  targetId: String,
  value: Number,             // 1（赞同）或 -1（反对）
  createdAt: Date
}
// 索引：user+targetType+targetId（唯一）
```

#### Report（举报）
```typescript
{
  reporter: 关联 User,
  targetType: String,
  targetId: String,
  targetAuthor: 关联 User,
  reason: String,            // spam/abuse/inappropriate/copyright/other
  description: String,
  status: String,            // pending/resolved/rejected
  handledBy: 关联 User,
  handleResult: String,
  handledAt: Date,
  createdAt: Date
}
// 索引：status, reporter, targetAuthor, createdAt
```

## 验收标准
- 所有模型在 prisma/schema.prisma 中独立定义，禁止单表JSON存储
- prisma migrate 成功执行，表结构正确创建
- 所有索引生效（通过 @@index 定义）
- 唯一约束正确生效（通过 @@unique 定义）
- 关联关系正确定义（@relation），外键约束生效
- 枚举类型在 Prisma 中通过 enum 定义
- 密码字段在服务层用 bcrypt 加密后再存入数据库
