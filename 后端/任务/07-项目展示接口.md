# 模块07：项目展示接口

> 重要：本模块接口路径必须与前端对应模块保持完全一致，任何变更需前后端同步更新。

## 模块概述
实现项目展示功能的后端接口，支持项目发布、评分、筛选等。

## 接口设计

```
GET    /api/v1/projects                 # 项目列表
GET    /api/v1/projects/:id             # 项目详情
POST   /api/v1/projects                 # 创建项目（需认证）
PUT    /api/v1/projects/:id             # 编辑项目（需认证）
DELETE /api/v1/projects/:id             # 删除项目（需认证）
POST   /api/v1/projects/:id/like        # 点赞
POST   /api/v1/projects/:id/favorite    # 收藏
POST   /api/v1/projects/:id/rate        # 评分（1-5星）
GET    /api/v1/projects/:id/ratings     # 评分列表
GET    /api/v1/projects/:id/related     # 相关项目
```

## 查询参数
```
GET /api/v1/projects?
  page=1&
  limit=20&
  category=xxx&
  techStack=react,node&       # 技术栈筛选（逗号分隔）
  status=completed&
  sort=latest|popular|rating&
  keyword=xxx
```

## 业务逻辑
- 评分系统：计算平均分，每用户只能评一次
- 技术栈筛选：支持多技术栈AND/OR查询
- 项目推荐：基于技术栈相似度
- 截图上传：最多6张，自动压缩和生成缩略图
