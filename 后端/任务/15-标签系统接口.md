# 模块15：标签系统接口

> 重要：本模块接口路径必须与前端模块12（标签分类）保持完全一致，任何变更需同步更新。

## 模块概述
实现面向普通用户的标签和分类查询、关注等接口。区别于管理后台接口中的标签管理（增删改合并），本模块只提供用户级读取和关注操作。

## 接口设计

### 1. 标签接口
```
GET    /api/v1/tags                        # 标签列表（支持分页、搜索）
GET    /api/v1/tags/:id                    # 标签详情
GET    /api/v1/tags/:id/contents           # 标签下的内容（帖子/博客/问题/项目/代码）
GET    /api/v1/tags/hot                    # 热门标签
POST   /api/v1/tags/:id/follow             # 关注标签（需认证）
DELETE /api/v1/tags/:id/follow             # 取消关注标签（需认证）
```

### 2. 分类接口
```
GET    /api/v1/categories                  # 分类列表（树形结构）
GET    /api/v1/categories/:id/contents     # 分类下的内容
```

## 查询参数
```
GET /api/v1/tags?
  page=1&
  limit=20&
  keyword=xxx&              # 搜索标签名
  sort=popular|latest|name  # 排序方式

GET /api/v1/tags/:id/contents?
  type=all|post|question|blog|snippet|project&  # 内容类型
  page=1&
  limit=20&
  sort=latest|popular
```

## 业务逻辑
- 标签列表支持按使用次数、关注人数、名称排序
- 热门标签取使用次数最多的前20个，Redis缓存10分钟
- 标签下的内容聚合多种类型，支持按类型筛选
- 关注/取关同步更新标签的followersCount
- 分类返回树形结构（包含子分类）
