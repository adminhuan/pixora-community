# 模块04：代码分享模块

> 重要：本模块的接口路径必须与后端对应模块保持完全一致，任何变更需前后端同步更新。

## 模块概述
提供类似GitHub Gist的代码片段分享功能，支持多文件代码分享、语法高亮、在线预览、一键复制等。

## 设计要求
- 科技蓝配色方案（坚决不使用紫色），代码编辑器区域采用暗色主题
- 调用 UI/UX PRO MAX skills 设计
- 代码区域需要高质量的语法高亮
- 禁止使用任何表情包和emoji

## 功能清单

### 1. 代码分享列表
- 按语言分类筛选
- 按热度/最新/收藏数排序
- 搜索代码片段
- 代码预览卡片（标题、语言标签、代码前几行预览、作者、星标数）

### 2. 创建代码分享
- 代码编辑器（支持语法高亮输入）
- 多文件支持（添加/删除文件标签页）
- 语言选择（自动检测 + 手动选择）
- 标题和描述输入
- 公开/私有设置
- 标签添加

### 3. 代码详情页
- 完整代码展示（行号 + 语法高亮）
- 多文件标签页切换
- 一键复制代码
- 原始文件下载
- 代码行数、字符数统计
- 点赞、收藏、Fork功能
- 评论区
- 版本历史（修改记录）

### 4. 代码运行（预留）
- 预留在线代码运行沙箱接口
- 支持常见语言运行预览

## 组件结构
```
src/pages/Code/
├── CodeList.tsx             # 代码分享列表
├── CodeDetail.tsx           # 代码详情
├── CreateCode.tsx           # 创建代码分享
├── EditCode.tsx             # 编辑代码分享
└── components/
    ├── CodeCard.tsx          # 代码卡片
    ├── CodeEditor.tsx        # 代码编辑器组件
    ├── CodeViewer.tsx        # 代码展示组件
    ├── LanguageSelect.tsx    # 语言选择器
    ├── FileTabs.tsx          # 多文件标签页
    └── CopyButton.tsx        # 复制按钮
```

## 接口对接
```
GET    /api/v1/snippets              # 代码列表
GET    /api/v1/snippets/:id          # 代码详情
POST   /api/v1/snippets              # 创建代码分享
PUT    /api/v1/snippets/:id          # 编辑
DELETE /api/v1/snippets/:id          # 删除
POST   /api/v1/snippets/:id/fork     # Fork
POST   /api/v1/snippets/:id/like     # 点赞
GET    /api/v1/snippets/:id/versions # 版本历史
```

## 依赖库
- Monaco Editor 或 CodeMirror 6：代码编辑器
- Prism.js：语法高亮
- highlight.js：语言自动检测

## 验收标准
- 支持至少20种主流编程语言的语法高亮
- 代码复制功能准确无误
- 多文件切换流畅
- 大段代码（1000行+）渲染不卡顿
- 代码编辑器输入体验接近VSCode
