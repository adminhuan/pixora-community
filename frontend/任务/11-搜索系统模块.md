# 模块11：搜索系统模块

> 重要：本模块的接口路径必须与后端对应模块保持完全一致，任何变更需前后端同步更新。

## 模块概述
全站搜索功能，支持多类型内容搜索、搜索建议、高亮匹配、高级筛选等。

## 设计要求
- 科技蓝配色方案（坚决不使用紫色）
- 调用 UI/UX PRO MAX skills 设计
- 搜索体验流畅，搜索建议响应迅速
- 禁止使用任何表情包和emoji

## 功能清单

### 1. 搜索入口
- 导航栏搜索框
- 搜索框获取焦点时展示搜索历史
- 输入时实时搜索建议（防抖处理）
- 快捷键唤起搜索（Ctrl/Cmd + K）

### 2. 搜索结果页
- 多类型标签页筛选（全部、帖子、问答、博客、代码、项目、用户）
- 搜索关键词高亮
- 结果排序（相关度、最新、最热）
- 高级筛选（时间范围、标签、作者）
- 结果分页
- 无结果提示与建议

### 3. 搜索建议
- 热门搜索词
- 搜索历史（本地存储）
- 实时搜索补全
- 清除搜索历史

## 组件结构
```
src/pages/Search/
├── SearchResults.tsx          # 搜索结果页
└── components/
    ├── SearchResultItem.tsx    # 搜索结果项
    ├── SearchFilters.tsx       # 高级筛选
    └── SearchTabs.tsx          # 类型标签页

src/components/Search/
├── SearchBar.tsx              # 全局搜索框
├── SearchSuggestions.tsx      # 搜索建议下拉
├── SearchHistory.tsx          # 搜索历史
└── SearchHighlight.tsx        # 关键词高亮组件

src/hooks/
└── useSearch.ts               # 搜索Hook（防抖、缓存）
```

## 接口对接
```
GET /api/v1/search?q=keyword&type=all&page=1     # 搜索
GET /api/v1/search/suggestions?q=keyword          # 搜索建议
GET /api/v1/search/hot                            # 热门搜索词
```

## 验收标准
- 搜索建议响应时间不超过300ms
- 关键词高亮准确
- 多类型搜索结果展示正确
- 搜索历史本地持久化
- 快捷键全局可用
