# 模块05：代码分享接口

> 重要：本模块接口路径必须与前端对应模块保持完全一致，任何变更需前后端同步更新。

## 模块概述
实现代码片段分享功能的后端接口，支持多文件、版本管理、Fork等。

## 接口设计

```
GET    /api/v1/snippets                 # 代码列表（分页、筛选）
GET    /api/v1/snippets/:id             # 代码详情
POST   /api/v1/snippets                 # 创建代码分享（需认证）
PUT    /api/v1/snippets/:id             # 编辑（需认证，仅作者）
DELETE /api/v1/snippets/:id             # 删除（需认证）
POST   /api/v1/snippets/:id/fork        # Fork代码（需认证）
POST   /api/v1/snippets/:id/like        # 点赞
POST   /api/v1/snippets/:id/favorite    # 收藏
GET    /api/v1/snippets/:id/versions    # 版本历史
GET    /api/v1/snippets/:id/raw/:file   # 获取原始代码文件
```

## 业务逻辑
- 创建/编辑时自动生成版本记录
- 语言自动检测（基于文件扩展名和内容）
- Fork操作：复制代码并关联原始来源
- 私有代码仅作者可见
- 代码内容大小限制（单文件最大100KB，总大小最大500KB）
