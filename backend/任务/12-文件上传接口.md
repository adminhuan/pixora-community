# 模块12：文件上传接口

> 重要：本模块接口路径必须与前端对应模块保持完全一致，任何变更需前后端同步更新。

## 模块概述
实现文件上传服务，支持图片、代码文件等的上传、存储、处理。

## 接口设计

```
POST   /api/v1/upload/image             # 上传图片
POST   /api/v1/upload/avatar            # 上传头像（自动裁剪）
POST   /api/v1/upload/images            # 批量上传图片
DELETE /api/v1/upload/:fileId            # 删除文件
```

## 上传配置

| 类型 | 最大大小 | 允许格式 | 处理 |
|------|---------|---------|------|
| 头像 | 2MB | jpg/png/gif/webp | 裁剪为正方形，生成多尺寸 |
| 帖子/博客图片 | 5MB | jpg/png/gif/webp | 压缩，生成缩略图 |
| 项目截图 | 5MB | jpg/png/webp | 压缩，生成缩略图 |
| 封面图 | 5MB | jpg/png/webp | 按比例裁剪 |

## 存储方案
```
开发环境：本地存储（/uploads目录）
生产环境：阿里云OSS / 七牛云（可配置切换）
```

## 业务逻辑
- 图片处理：使用sharp库进行压缩、裁剪、格式转换
- 头像处理：裁剪为200x200、100x100、50x50三个尺寸
- 文件命名：年/月/日/uuid.ext 目录结构
- 文件去重：基于文件hash判断是否已存在
- 安全检查：文件类型校验（检查magic bytes，不仅仅依赖扩展名）
- CDN：生产环境返回CDN地址
- 清理：定期清理未被引用的孤儿文件
