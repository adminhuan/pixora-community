# 模块01：项目初始化与基础架构

## 模块概述
搭建Node.js + Express + TypeScript后端项目的基础框架，配置开发环境、中间件、错误处理等。

## 技术选型
- 运行环境：Node.js 18+
- 框架：Express 4.x
- 语言：TypeScript 5.x
- 数据库：PostgreSQL + Prisma ORM
- 缓存：Redis
- 日志：Winston
- 进程管理：PM2

## 功能清单

### 1. 项目初始化
- TypeScript配置（tsconfig.json）
- ESLint + Prettier代码规范
- 环境变量管理（dotenv，区分开发/测试/生产）
- 启动脚本配置（开发热重载nodemon、生产构建）

### 2. Express应用配置
- CORS跨域配置
- Body解析（JSON、URL编码、multipart）
- 静态文件服务
- 请求日志（Morgan）
- 安全中间件（Helmet、rate-limit、hpp）
- 压缩中间件（compression）

### 3. 数据库连接
- PostgreSQL连接管理（Prisma ORM）
- Redis连接管理（ioredis）
- Prisma Client 初始化与连接池配置
- 断线重连机制

### 4. 错误处理
- 全局错误处理中间件
- 自定义错误类（AppError）
- HTTP状态码规范
- 统一响应格式

### 5. 日志系统
- 分级日志（error/warn/info/debug）
- 日志文件轮转
- 请求日志记录

### 6. API版本管理
- /api/v1 路由前缀
- API文档（Swagger/OpenAPI）

## 统一响应格式
```typescript
// 成功响应
{
  success: true,
  data: {},
  message: "操作成功"
}

// 错误响应
{
  success: false,
  error: {
    code: "VALIDATION_ERROR",
    message: "参数验证失败",
    details: []
  }
}

// 分页响应
{
  success: true,
  data: [],
  pagination: {
    page: 1,
    limit: 20,
    total: 100,
    totalPages: 5
  }
}
```
