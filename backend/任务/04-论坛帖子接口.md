# 模块04：论坛帖子接口

> 重要：本模块接口路径必须与前端对应模块保持完全一致，任何变更需前后端同步更新。

## 模块概述
实现论坛帖子的增删改查、分类筛选、排序、置顶精华等后端接口。

## 接口设计

```
GET    /api/v1/posts                    # 帖子列表（分页、筛选、排序）
GET    /api/v1/posts/:id                # 帖子详情
POST   /api/v1/posts                    # 创建帖子（需认证）
PUT    /api/v1/posts/:id                # 编辑帖子（需认证，仅作者）
DELETE /api/v1/posts/:id                # 删除帖子（需认证，作者或管理员）
POST   /api/v1/posts/:id/like           # 点赞/取消点赞
POST   /api/v1/posts/:id/favorite       # 收藏/取消收藏
GET    /api/v1/posts/:id/related        # 相关帖子推荐
```

## 查询参数
```
GET /api/v1/posts?
  page=1&                    # 页码
  limit=20&                  # 每页条数
  category=xxx&              # 分类ID
  tag=xxx&                   # 标签ID
  sort=latest|hot|featured&  # 排序方式
  keyword=xxx&               # 搜索关键词
  author=xxx                 # 作者ID
```

## 业务逻辑
- 创建帖子：Markdown内容转HTML存储，提取标签，自动生成摘要
- 浏览计数：使用Redis防止重复计数（同一用户10分钟内只算一次）
- 点赞：Redis缓存 + 定时同步到MongoDB
- 相关推荐：基于标签和分类的相似度匹配
- 帖子列表：热门排序综合考虑浏览量、点赞数、评论数和时间衰减
